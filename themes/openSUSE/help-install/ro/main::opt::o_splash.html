Opțiuni kernel: splash (ecran de pornire)

Ecranul de start este imaginea afișată în timp ce pornește sistemul.

  • splash=0 -- Ecranul de pornire este dezactivat. Acest lucru poate fi util pentru monitoarele foarte vechi sau în cazul apariției unei erori.
  • splash=verbose -- Activează ecranului de pornire, mesajele kernel și boot sunt afișate în continuare.
  • splash=silent -- Activează ecranul de pornire, dar fără mesaje. În schimb se afișează o bară de progres.
