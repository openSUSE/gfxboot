Ρυθμίσεις Εκκίνησης

There are two types of boot options available. First, there are options that affect the installer. Second, there are kernel options. Some of the more common options are:

a) installer options

  • εγκατάσταση -- select an installation source
  • επιλογές δικτύου -- the network options
  • επιλογές vnc -- options for installation via VNC

b) kernel options

  • αρχική οθόνη -- influence the behavior of the splash screen
  • apm -- toggle power management
  • acpi -- advanced configuration and power interface
  • ide -- control the IDE subsystem
