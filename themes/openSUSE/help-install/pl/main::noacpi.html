Instalacja (bez ACPI)

Dzisiejszy sprzęt zazwyczaj wymaga ACPI do kontrolowania przerwań. APCI całkowicie zastępuje stary system APM.

Jeżeli podczas uruchamiania systemu występują problemy, należy wybrać opcję Instalacja (bez ACPI). Znanymi błędami sprzętowymi występującymi przy używaniu ACPI są:

  • jądro zawiesza się przy uruchamianiu
  • karty PCI nie zostały wykryte lub zinicjalizowane prawidłowo

Można również spróbować opcji rozruchowej Test oprogramowania firmowego i sprawdzić wyniki testu ACPI.
