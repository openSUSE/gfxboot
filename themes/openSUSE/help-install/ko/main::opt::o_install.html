설치 옵션: install

기본적으로, 내부 CD-ROM의 설치 소스를 검색합니다. 네트워크 인스톨을 위해서는설치 옵션을 선택합니다. 설치 진행이 가능한 프로토콜은

  • FTP
  • NFS
  • HTTP

구문 사용은 표준 URL 같은 것만을 사용합니다. 예를 들어, 192.168.0.1 에서 서버를 찾고 하고자 하는 NFS 기반 설치를 이 서버의 디렉토리 /install 에서 할 수 있습니다. 지정되는 소스는 다음과 같습니다:

  • install=nfs://192.168.0.1/install

네트워크 카드는 둘중 하나를 사용하여 구성됩니다. dhcp 또는 네트워크 옵션 에서 자신의 설명에 따라 매개 변수를 지정해야 합니다.
