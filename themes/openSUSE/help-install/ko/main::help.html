도움말 시스템 사용

부트로더 온라인 도움말은 context sensitive 입니다. 선택한 메뉴 아이템에 대한 정보를 얻거나, 부트 옵션을 편집 할 때 옵션에 커서가 올라가있으면 거기에 대한 정보를 조회 할 수 있습니다.

키

  • 위 화살표: 앞의 링크를 하이라이트
  • 아래 화살표: 다음 링크를 하이라이트
  • 왼쪽 화살표, 백스페이스: 이전 주제로 되돌아가기
  • 오른쪽 화살표, 엔터, 스페이스: 다음 링크를
  • 페이지 업: 한장 위로
  • 페이지 다운: 한장 아래로
  • 처음: 첫장으로
  • End: 마지막 장으로
  • Esc: 도움말 떠남
