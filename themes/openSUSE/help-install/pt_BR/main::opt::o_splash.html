Opções do kernel: splash

A tela de inicialização é a imagem mostrada durante a inicialização do sistema.

  • splash=0 -- A tela de inicialização é desativada. Isto poderá ser útil com monitores muito antigos, ou se ocorrerem erros.
  • splash=verbose -- Ativa a tela de inicialização, as mensagens do kernel e de boot ainda são mostradas.
  • splash=silent -- Ativa a tela de inicialização, mas sem mensagens. Em vez disto, é exibida uma barra de progresso.
