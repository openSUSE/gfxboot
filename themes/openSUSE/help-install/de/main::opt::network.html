Installiereroptionen: Netzwerkoptionen

Es ist möglich die Netzwerkschnittstelle sofort zu konfigurieren. Die Hardwarewird später durch YaST2 erkannt. Der minimale Satz an Optionen, um IhreNetzwerkkarte zu konfigurieren besteht aus Host-IP und Netzmaske.Beispiel:

  • hostip=192.168.0.10 netmask=255.255.255.0

oder in einer kürzeren Form:

  • hostip=192.168.0.10/24

Wenn Sie netzwerkbasierte Installation bestimmt haben, aber diese beiden Optionen nichtbestimmen, versucht der Installierer die Netzwerkschnittstelle mit dhcpzu konfigurieren. Wenn Sie eine Standardschnittstelle benötigen, bestimmen Sie diesmit der Option Gateway. Beispiel:

  • gateway=192.168.0.8
