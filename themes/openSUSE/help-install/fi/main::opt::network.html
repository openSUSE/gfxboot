Asentimen valinnat: Verkon valinnat

Verkkoliitäntä on mahdollista määrittää nyt. Laitteisto tunnistetaan myöhemmin YaST toimesta. Pakolliset valinnat verkon määrittämiseksi ovat IP-osoite ja aliverkon peite. Esim:

  • hostip=192.168.0.10 netmask=255.255.255.0

tai lyhyemmässä muodossa:

  • hostip=192.168.0.10/24

Jos määritit asennus verkosta:n etkä määritä molempia näistä valinnoista, asennusohjelma yrittää määrittää verkkoliitännän dhcp:llä. Jos tarvitset oletusyhdyskäytävän, määritä se valinnalla gateway. Esim:

  • gateway=192.168.0.8
