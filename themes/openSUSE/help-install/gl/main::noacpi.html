Instalación -- ACPI Deshabilitado

O hardware actual normalmente precisa ACPI (Advanced Configuration and Power Interface) para controlar o manexo de interrupcións. ACPI substitúe totalmente ó antigo sistema APM

Seleccione Instalación -- ACPI Deshabilitado se ten problemas durante o inicio do kernel. Os problemas coñecidos en máquinas que teñen problemas con ACPI son:

  • o kernel conxélase ao arrincar
  • As tarxetas PCI non se detectan ou se inicializan axeitadamente

    • Tamén podería probar a opción de arrinque Proba de Firmware e mirar os resultados da proba de validación de ACPI.
