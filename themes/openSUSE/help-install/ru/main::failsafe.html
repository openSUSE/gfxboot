Установка - безопасные параметры настройки

Выберите Установка - безопасные параметры настройки если у вас подвисает система при установке или возникают непонятные ошибки. Этот параметр отключает DMA для IDE устройств и все особенности управления питанием. Смотри также параметры ядра apm, acpi и ide.
