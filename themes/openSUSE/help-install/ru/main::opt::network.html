Параметры установки: Параметры сети

Можно настроить сетевой интерфейс прямо сейчас. Оборудование будет определено позже YaST2. Минимальный набор параметров вашей сетевой карты состоит из IP адреса узла и сетвой маски. Для примера:

  • hostip=192.168.0.10 netmask=255.255.255.0

или в коротком виде:

  • hostip=192.168.0.10/24

Если вы указали сетевая установка и не указали оба эти параметра, установщик попытается настроить сетевой интерфейс с dhcp. Если вам необходим шлюз по умолчанию, укажите это с параметром шлюз. Например:

  • gateway=192.168.0.8
