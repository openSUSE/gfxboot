開機載入器說明

歡迎使用 SUSE LINUX 9.2

請使用此功能表，選取想要使用的功能。 若不知如何在此說明系統中瀏覽，請按 F1 鍵以進入此說明系統的介紹。 以下為此功能表的主要功能：

從硬碟開機： 此選項將不會對系統做任何變動。 只會啟動先前安裝的作業系統。

安裝： 此安裝模式適用於大部分電腦。 若開機過程中系統停止不動，或是偵測磁碟控制器或網路卡等硬體元件時發生問題，請使用下列安裝選項。

安裝 -- 關閉 ACPI： 目前市面上販售的許多電腦，對於 ACPI 的實作並不完整，或是實作有誤。 此選項將停用核心對於 ACPI 的支援，但仍啟用眾多提高效能的功能，例如，IDE 硬碟的 DMA 功能。

安裝 -- 安全設定： 若使用安裝模式未能成功安裝系統，可考慮改用此選項。

救援系統： 此開機影像將在 RAM 中啟動小型的 Linux 系統。 若系統無法正常啟動，可使用此選項。 系統開機後，以 root 身份登入。

記憶體測試： 記憶體測試不僅可用來檢查新安裝的記憶體模組， 也可針對電腦系統的大部分元件進行壓力測試，並可能指出硬體問題。

開機選項： 此開機選項可能完全變更您的系統功能。 這些是屬於核心的設定。

F1 說明： 此說明系統可根據使用者的工作內容來提供協助。 說明系統將依照開機畫面中的使用元素，顯示不同的畫面。 此外，也提供了關於此說明系統的介紹。

F2 視訊模式： 請在此選擇安裝時的螢幕解析度。 若使用圖形介面安裝時發生問題，可考慮使用文字模式。

F3 安裝來源： 選擇安裝來源。

F4 語言： 設定開機載入器所用的語言與鍵盤配置。

F5 驅動程式更新： 若是使用較新型的電腦，可能需要更新驅動程式，才能安裝系統。

驅動程式更新

驅動程式更新通常是在磁片上，儲存了安裝所需的硬體新版驅動程式或修補程式。

安裝

選取安裝以便開始進行預設安裝。 輸入的開機選項將在啟動時使用。 此項目可啟用常見硬體的許多功能。



返回開始頁面

從硬碟開機

選取開機已安裝的系統，啟動已安裝在本機硬碟上的系統。 此系統必須正確安裝，因為只會啟動第一顆硬碟的 MBR (主開機記錄)。 第一顆硬碟的的裝置 ID 是由電腦的 BIOS 所提供。

若您忘記從光碟機中取出 CD 或 DVD，而要從硬碟啟動電腦，可使用此選項。



返回開始頁面

安裝 -- 關閉 ACPI

在 SuSE Linux 中，穩定版 2.4 核心已可使用開發版 2.5 核心的 ACPI 支援功能。 部分較新的硬體，有時可能需要 ACPI 來控制中斷要求處理。 ACPI 可完整替代舊式的 APM 系統。

若在核心開機時出現問題，請選取安裝 -- 關閉 ACPI 。 電腦 ACPI 不正常時，可能會發生下列已知問題：

* 開機時核心停止不動
* 無法正確偵測或啟始化 PCI 介面卡



返回開始頁面

安裝 -- 安全設定

若在安裝時系統當機，或發生無法重現的錯誤，請選取安裝 -- 安全設定。 此選項將停用 IDE 磁碟機的 DMA 功能，以及所有電源管理功能。 請參閱 apm、acpi 與 ide 的核心選項。

返回開始頁面

救援系統

救援系統為使用 RAM 磁碟機的小型系統。 利用此一小型系統，可變更已安裝系統的任何設定。 由於此系統僅提供低階工具，因此僅供進階使用者所用。



返回開始頁面

記憶體測試

記憶體測試可針對系統的硬體進行壓力測試。 記憶體測試的主要目的是偵測故障的 RAM，但也可用來針對系統的許多元件進行壓力測試。
雖然記憶體測試可找出大部分有問題的記憶體，但是，若偵測不到問題，並不保證記憶體必然正常。

開機選項

有兩類可用的開機選項。 第一類為影響安裝程式的選項。 第二類為核心選項。 以下為部分常用的選項：

a) 安裝程式選項

install -- 選取安裝來源
network 選項 -- 網路選項
vnc 選項 -- 經由 VNC 安裝的選項

b) 核心選項

splash -- 指定開機顯示畫面
apm -- 切換電源管理
acpi -- 進階組態與電源介面
ide -- 控制 IDE 子系統


返回開始頁面

安裝程式選項： install

根據預設，將在本機 CD-ROM 上搜尋安裝來源。 若是進行網路安裝，請選取 install 選項。 可用的安裝協定為
* FTP
* NFS
* HTTP
所用的語法與標準的 URL 相同。 例如，若伺服器的 IP 位址為 192.168.0.1，您要從伺服器的 /install 目錄利用 NFS 來進行安裝，則請使用下列語法來指定安裝來源：

install=nfs://192.168.0.1/install

網路卡必須以 dhcp 來設定，或是必須以 network 選項 所述的方式自行指定參數。


返回開機選項

核心選項： splash

開機顯示畫面為系統開機時所顯示的圖片。

splash=0

關閉開機顯示畫面。 若是使用很舊的螢幕，或是發生任何錯誤，可考慮使用此選項。

splash=verbose

啟用開機顯示畫面，同時也一併顯示核心與開機訊息。

splash=silent

啟用開機顯示畫面，但不顯示訊息。 顯示進度列供使用者參考。

返回開機選項

安裝程式選項： Network 選項

您可以在此設定網路介面。 YaST2 稍後將會偵測硬體。 設定網路卡的最小一組選項包含主機 IP 位址與網路遮罩。 例如：

hostip=192.168.0.10 netmask=255.255.255.0

或是以簡寫的方式：

hostip=192.168.0.10/24

若您指定以 network 為基礎的安裝，但未指定上述兩項參數，安裝程式將嘗試利用 dhcp 來設定網路介面。 若需要使用預設閘道，請以 gateway 選項來指定。 例如：

gateway=192.168.0.8


返回開機選項

安裝程式選項： vnc

若要使用 VNC 安裝，請指定 vnc 與 vncpassword 參數：

vnc=1 vncpassword=example

VNC 伺服器將會啟動。您可以從遠端系統的 VNC 用戶端來控制 YaST2。


返回開機選項。

核心選項： apm

APM 為目前電腦所用的兩種電源管理策略之一。 APM 主要用於筆記型電腦，可提供休眠 (suspend to disk) 等功能，同時也可負責停電時關閉電腦的功能。 APM 必須搭配適當的 BIOS。 若 BIOS 故障，APM 只能提供有限的功能，或甚至使得電腦無法運作。 在此狀況下，可利用此參數來關閉 APM 功能

apm=off -- 關閉 APM 全部的功能

部分新型電腦可能採用較新的 ACPI。

返回開機選項

核心選項： acpi

ACPI (進階組態與電源介面) 為業界標準，定義作業系統與 BIOS 之間的電源與組態管理介面。 根據預設，若偵測到 BIOS 為西元 2000 年之後的版本，將會啟用 acpi。以下是控制 ACPI 功能常用的幾個參數：

pci=noacpi -- 不使用 ACPI 來遞送 PCI 中斷要求 acpi=oldboot -- 只啟用與開機有關的部分 ACPI
acpi=off -- 關閉 ACPI 全部的功能
acpi=force -- 即使 BIOS 為西元 2000 年前的版本，仍舊啟用 ACPI

特別是在新的電腦上，可取代舊式的 apm 系統。

返回開機選項

核心選項： ide

IDE 與 SCSI 的差異在於 IDE 比 SCSI 更廣泛地運用在大部分桌上型電腦。 為了避免 IDE 系統相關的硬體問題，請使用下列核心參數：

ide=nodma -- 關閉 IDE 磁碟機或光碟機的 DMA 功能


返回開機選項。
